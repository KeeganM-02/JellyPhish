# JellyPhish

**JellyPhish** is an experimental Chrome extension designed to extract email content from webmail clients for phishing detection research. The extension currently focuses on Gmail and Outlook web interfaces and lays the groundwork for analyzing email content, links, and sender information.  

## Features (Planned)

- Extracts subject, sender, body text, and links from emails.
- Compatible with Gmail and Outlook webmail.
- Stores email metadata locally for analysis.
- Future integration of machine learning models to identify potential phishing emails.  

## Installation (Developer Mode)

1. Clone or download this repository.  
2. Open Chrome and navigate to `chrome://extensions/`.  
3. Enable **Developer mode** (top right).  
4. Click **Load unpacked** and select the project folder.  
5. The extension will appear in the toolbar.  

> Note: Currently, this is an experimental development version. Functionality may be limited, and it is intended for testing and research purposes only.
